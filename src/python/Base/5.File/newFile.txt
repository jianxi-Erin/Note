25.但没办法，我不是凝光，做不出绝对完美的交易。

有些人眼中，人情很值钱，需要好好规划着用，这是把人情视为筹码。

又有些人眼中，人情不仅一钱不值，还会带来繁重的往来世故，这是把人情视为负担。

但在我看来，人情这东西，说到底还是人心与人心之间的连结。

几千年来的璃月人，就是通过你麻烦我，我拜托你这种所谓人情的小事，建立起了彼此之间的连结。

如此一来，一个人就不再只是一个人了，在他摔倒的时候，会有看不见的连结之网托着他重新站起来。

我想哪怕有一天，群魔降世，律法崩坏，璃月重回战争年代。

这种连结也会护佑人们走出黑暗吧。——北斗
