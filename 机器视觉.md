### OpenCV（Open Source Computer Vision Library）是一个开源计算机视觉库，提供了丰富的功能和方法来处理图像、视频和摄像头流。以下是OpenCV中常用的图像、视频和摄像头处理方法及其参数的简要解释，以表格的形式展示。
### 1. 图像处理方法

| **函数**                           | **描述**                                                 | **主要参数**                                                                                                                                                           |
|------------------------------------|----------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| `cv2.imread(filename, flags)`      | 读取图像文件                                              | - `filename`：图像文件路径<br>- `flags`：读取模式（0=灰度，1=彩色，-1=不改变原始格式）                                                                                   |
| `cv2.imwrite(filename, img)`       | 保存图像到文件                                            | - `filename`：保存文件的路径<br>- `img`：要保存的图像数据                                                                                                              |
| `cv2.imshow(winname, img)`         | 在窗口中显示图像                                          | - `winname`：窗口名称<br>- `img`：要显示的图像                                                                                                                         |
| `cv2.resize(src, dsize, fx, fy)`   | 调整图像大小                                              | - `src`：源图像<br>- `dsize`：输出图像尺寸<br>- `fx`：宽度比例缩放<br>- `fy`：高度比例缩放                                                                               |
| `cv2.cvtColor(src, code)`          | 转换图像的颜色空间                                        | - `src`：源图像<br>- `code`：颜色转换代码（如`cv2.COLOR_BGR2GRAY`、`cv2.COLOR_BGR2RGB`）                                                                                |
| `cv2.GaussianBlur(src, ksize, sigmaX)` | 对图像进行高斯模糊处理                                  | - `src`：源图像<br>- `ksize`：核大小<br>- `sigmaX`：高斯核在X方向的标准偏差                                                                                             |
| `cv2.threshold(src, thresh, maxval, type)` | 对图像进行阈值处理                                    | - `src`：源图像<br>- `thresh`：阈值<br>- `maxval`：最大值<br>- `type`：阈值类型（如`cv2.THRESH_BINARY`）                                                                |
| `cv2.Canny(image, threshold1, threshold2)` | 应用Canny边缘检测                                      | - `image`：输入图像<br>- `threshold1`：较低的阈值<br>- `threshold2`：较高的阈值                                                                                         |
| `cv2.findContours(image, mode, method)` | 检测图像中的轮廓                                        | - `image`：输入二值图像<br>- `mode`：轮廓检索模式（如`cv2.RETR_TREE`）<br>- `method`：轮廓逼近方法（如`cv2.CHAIN_APPROX_SIMPLE`）                                       |
| `cv2.drawContours(image, contours, contourIdx, color, thickness)` | 在图像上绘制轮廓     | - `image`：输入图像<br>- `contours`：轮廓点列表<br>- `contourIdx`：绘制特定轮廓的索引<br>- `color`：轮廓颜色<br>- `thickness`：轮廓线条的粗细                          |

### 2. 视频处理方法

| **函数**                                    | **描述**                                                     | **主要参数**                                                                                                                                               |
|---------------------------------------------|--------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------|
| `cv2.VideoCapture(filename)`                | 打开视频文件或摄像头                                          | - `filename`：视频文件路径或摄像头设备ID                                                                                                                   |
| `cv2.VideoCapture.isOpened()`               | 检查视频文件或摄像头是否成功打开                              | 无                                                                                                                                                         |
| `cv2.VideoCapture.read()`                   | 读取视频帧                                                   | - 返回值：布尔值表示是否成功读取帧，图像数据表示读取的帧                                                                                                   |
| `cv2.VideoCapture.release()`                | 释放视频捕获对象                                              | 无                                                                                                                                                         |
| `cv2.VideoWriter(filename, fourcc, fps, frameSize)` | 保存视频到文件                                              | - `filename`：保存的视频文件路径<br>- `fourcc`：编码器代码（如`cv2.VideoWriter_fourcc(*'XVID')`）<br>- `fps`：帧率<br>- `frameSize`：帧尺寸（宽, 高）        |
| `cv2.VideoWriter.write(frame)`              | 向视频文件写入帧                                              | - `frame`：要写入的图像帧                                                                                                                                |
| `cv2.VideoWriter.release()`                 | 释放视频写入对象                                              | 无                                                                                                                                                         |

### 3. 摄像头处理方法

| **函数**                                 | **描述**                                                   | **主要参数**                                                                                                                                                   |
|------------------------------------------|------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------|
| `cv2.VideoCapture(0)`                    | 打开默认摄像头                                             | - `0`：设备ID，通常是摄像头的索引号。`0`为默认摄像头，其他摄像头可以使用1, 2等                                                                                  |
| `cv2.VideoCapture.get(propId)`           | 获取摄像头的属性                                           | - `propId`：摄像头属性ID（如`cv2.CAP_PROP_FRAME_WIDTH`、`cv2.CAP_PROP_FRAME_HEIGHT`、`cv2.CAP_PROP_FPS`）                                                       |
| `cv2.VideoCapture.set(propId, value)`    | 设置摄像头的属性                                           | - `propId`：摄像头属性ID<br>- `value`：属性值                                                                                                                  |
| `cv2.waitKey(delay)`                     | 等待键盘输入                                               | - `delay`：等待的时间，以毫秒为单位。`0`表示无限等待                                                                                                            |
| `cv2.destroyAllWindows()`                | 关闭所有OpenCV窗口                                         | 无                                                                                                                                                             |
| `cv2.destroyWindow(winname)`             | 关闭指定名称的窗口                                         | - `winname`：窗口名称                                                                                                                                          |
| `cv2.VideoCapture.release()`             | 释放摄像头                                                 | 无                                                                                                                                                             |

### 4. 辅助方法

| **函数**                                 | **描述**                                                   | **主要参数**                                                                                       |
|------------------------------------------|------------------------------------------------------------|---------------------------------------------------------------------------------------------------|
| `cv2.waitKey(delay)`                     | 等待特定的键盘事件并返回ASCII码                            | - `delay`：等待的时间，单位为毫秒，0表示无限等待                                                  |
| `cv2.destroyAllWindows()`                | 销毁所有创建的窗口                                         | 无                                                                                                 |
| `cv2.getStructuringElement(shape, ksize)`| 获取结构元素，用于形态学操作                                | - `shape`：形状（如`cv2.MORPH_RECT`）<br>- `ksize`：内核大小（宽, 高）                             |
